# 🏗 Architecture – UrbanThreads

## MERN Stack
- **Frontend:** React + Vite + TailwindCSS
- **Backend:** Node.js + Express.js
- **Database:** MongoDB via Mongoose

## Folder Structure
```
UrbanThreads/
├── client/ (React App)
│   ├── src/
│   └── tailwind.config.js
├── server/
│   ├── models/
│   ├── routes/
│   ├── controllers/
│   └── .env
```

## Communication
- RESTful APIs between frontend and backend
- MongoDB stores product, user, and order data
